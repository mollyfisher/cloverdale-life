Cloverdale's Code Enforcement Setup, Now With Slightly More Structure

The city has no Code Enforcement Department, no dedicated officer, no consultant on retainer. Wednesday's staff report outlines three modest steps to professionalize a function held together by one Associate Planner and a job description that doesn't formally include the work.

May 8, 2026

Here is something most residents don't realize about how Cloverdale handles code complaints: there is no Code Enforcement Department. No dedicated officer. No consultant on retainer. There is, technically, an Associate Planner whose job description does not formally include code enforcement, but who has been doing it anyway for several years.

We are a small town. This is a small-town arrangement.

Code enforcement covers a wider range of complaints than most people picture — everything from abandoned vehicles on public streets to construction without permits to property maintenance issues that affect neighbors. Each type ends up under a different department's authority. State law and the Municipal Code lay out specific procedures and timelines for each. None of which fits neatly into one person's regular workload.

The Three Steps in Wednesday's Report

City Manager Kevin Thompson's report outlines three modest improvements.

1. A Coordinated Intake System

A new "Code Enforcement" permit type has been added to the city's existing permitting portal. When a complaint comes in, the system automatically notifies whichever combination of Fire, Building, Planning, Police, and Public Works needs to look at it. Abandoned car on the street? Police. Construction without permits? Building. Tall grass next to a wood pile in fire season? Fire. The portal also tracks the procedural steps and timelines that state law requires, which is the part of code enforcement that gets cities sued when it's done sloppily.

2. A Regional Contract for the Hard Cases

A regional Request for Proposals is in development with the cities of Cotati, Windsor, and Sonoma to share a contracted firm for the genuinely difficult cases — the ones that need formal notices, hearings, and a defensible paper trail. Cloverdale's share would run about $20,000, and only if a qualified firm responds to the RFP. The contracted services would be reserved for particularly difficult or complex cases, not routine work.

3. A Title Change That Matches Reality

Staff is also considering reclassifying the Associate Planner position to Senior Planner, with code enforcement formally written into the job description. The salary increase would be roughly 6.5%. The Council is not voting on this Wednesday — it'll come back as a separate item at a future meeting — but the report flags the direction.

What It Would Cost

Two line items would need to be funded if the Council wants to move forward:

  • Reclassification of the Associate Planner to Senior Planner — about a 6.5% salary increase, funding source not yet specified.
  • Up to $20,000 from the General Fund for the regional code enforcement contract, with actual spending depending on case volume.

For context, $20,000 is roughly the same scale as several other discretionary line items the Council is considering this budget cycle — comparable to the Resilient Cloverdale program, the Senior Center support increase, or the West Hills annexation consultant. Modest, but real.

What Happens Wednesday

The Council's job Wednesday is to receive the report and provide comments. No formal action. The reclassification, the contract, and any expanded staffing all come back later — most likely as part of the FY 26–27 budget the Council is currently shaping. If you have an opinion about how aggressive the city's code enforcement should be, or what kinds of complaints should get prioritized, this is a useful meeting to put it on the record.

Sources: City of Cloverdale Agenda Report for Item H.1, "Receive Report on Code Enforcement Services," May 13, 2026, prepared by City Manager Kevin Thompson.
