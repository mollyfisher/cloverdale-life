Code Enforcement Report on the May 13 Agenda

Item H.1 outlines three changes to how Cloverdale handles code enforcement complaints, including a new permit-portal workflow, possible participation in a regional contract for complex cases, and a possible reclassification of the Associate Planner position.

May 8, 2026

The Council requested an update on code enforcement services. Item H.1 on the May 13 agenda is a report from City Manager Kevin Thompson responding to that request. The recommended action is to receive the report and provide comments. No formal action is proposed.

What the Report Says About Current Operations

Per the agenda report:

"The City does not have dedicated Code Enforcement staff or consultants. Instead, City staff perform various aspects of Code Enforcement in addition to their regular duties, which can place demands on limited staff time."

The report describes code enforcement cases as varying significantly by complaint type and responsible department. It gives two examples: an abandoned vehicle on a public street is generally handled by the Police Department; construction occurring without permits is generally handled by the Building Department.

Three Changes Described in the Report

1. New permit type in the existing portal

Staff has created a new permit type in the City's existing permitting portal titled "Code Enforcement." When a complaint is received, the system automatically notifies "all potentially involved departments — including Fire, Building, Planning, Police, and Public Works." The report states that the new system will enable staff to "better track those steps and applicable timelines" required by the Municipal Code and state law.

2. Regional RFP under consideration

The report describes a regional Request for Proposals being developed with the Cities of Cotati, Windsor, and Sonoma for supplemental code enforcement services. Per the report:

"The City's estimated participation cost would be approximately $20,000 and is contingent upon a qualified firm responding to the RFP. These contracted services would only be utilized for particularly difficult or complex cases."

The $20,000 figure also appears in the May 5 budget workshop presentation under "Planning Initiatives," with proposed funding from the General Fund in FY 26–27.

3. Possible reclassification of the Associate Planner position

The report states that the Associate Planner is currently responsible for handling code enforcement cases, "however, the position's existing job description does not formally include code enforcement duties, despite these responsibilities having been performed for several years." Staff is considering amending the job description and reclassifying the position to Senior Planner. Per the agenda report, this reclassification "will be formally discussed at the future council meeting."

Financial Impact as Stated in the Report

  1. "Additional funding would be required to reclassify the Associate Planner position to Senior Planner, resulting in an estimated salary increase of approximately 6.5 percent over the current salary."
  2. "Additional funding in the amount of $20,000 from the General Fund may be allocated to contract for code enforcement services for difficult or complex cases. Actual use of these funds would depend on workload and case volume."

The report lists the CEQA determination as "Not a Project."

What Action Is Being Asked For Wednesday

Per the staff recommendation, Council is asked to receive the report and provide comments. The reclassification will return to a future meeting as a separate item. The $20,000 regional contract participation is reflected in the FY 26–27 draft budget as a discretionary item.

Source: City of Cloverdale Agenda Report for Item H.1, "Receive Report on Code Enforcement Services," May 13, 2026, prepared by City Manager Kevin Thompson. Budget context from the May 5, 2026 budget workshop presentation.
